# タイヤサイズ別 売上集計サイト（MVP）

固定レイアウトの「単品別売上集計リスト」PDFから、タイヤサイズと売上数量をOCRで読み取り、サイズ別に合算する静的Webアプリです。

## 主な機能

- PDFを複数まとめて読み込み
- ページごとにOCR
- `155/65R14` 形式のタイヤサイズを抽出
- 売上数量列を固定位置から取得
- サイズ別の本数集計
- OCR結果の手修正・行削除
- 集計CSV／明細CSVの出力
- PDFはブラウザ内で処理し、独自サーバーには送信しない構成

## 起動方法

### GitHub Pages

1. 新しいリポジトリを作成
2. `index.html`、`styles.css`、`app.js` をアップロード
3. Settings → Pages → `main` / `root` を選択
4. 表示されたURLを開く

### PCで確認

フォルダ内で次を実行し、表示されたURLをブラウザで開きます。

```bash
python -m http.server 8000
```

通常は `http://localhost:8000` です。`index.html` を直接ダブルクリックするより、簡易サーバー経由の方が安定します。

## 読み取り設定

添付サンプルでは、売上数量列の中心がページ幅の約64.3%にあります。帳票レイアウトが同じなら初期設定のままで動作します。

別レイアウトで数量を拾えない場合は、画面の「読み取り設定」で数量列の位置と許容幅を調整します。

## 使用ライブラリ

- PDF.js 3.11.174
- Tesseract.js 5

CDN経由で読み込むため、初回利用時とOCR実行時はインターネット接続が必要です。
